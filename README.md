# 本地预览
```shell
  git clone https://github.com/wb421768544/component-show.git
  cd component-show
  npm install
  npm start
```

# 使用wb-react-ui组件库
```shell
  npm install --save wb-react-ui
```
具体接口请移步[展示页](http://wangbing520.club/wb-ui)
